# 武软e学堂刷客， 武软公开课刷课
## 如何使用刷课
    1. 先登录自己的  武软公开课
    2. 点击开始学习（前提把课选好了）
    3. f12 打开控制台，输入 index.js里面的代码（打开浏览器控制台，有些笔记本是 fn + f12）
    4. 稍微等几秒在去看看  学习进度 应该就完成了（已测：大数据课成刷不了）

## 考试
    1. 找到测验考试，直接点提交
    2. 进入 学习进度=>点 提交次数=> 查看成绩就能看到答案
    3. 在把 查看成绩上面的  url 里面的 &view=1 删掉 就可以再次考试了（答案cv完事）
